1. Скрипт db_backup.sh выполняет резервное копирование баз СУБД MySQL.
2. Перечень баз для резервного копирования и параметры подключения к базам указываются в скрипте в переменной dbList в виде списка значений разделённых пробелами
3. Каждое значение списка содержит имя базы данных, пользователя, пароль и IP адрес разделённые символом @
4. Все сообщения по ходу выполнения скрипта записываются в файл db_backup.log
5. Подразумевается что директория /backups существует.
